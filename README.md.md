📊 Employee Management System (Python OOP)
A simple Employee Management System built using Python and Object-Oriented Programming (OOP) concepts.
This project demonstrates how classes, inheritance, and encapsulation work in Python.

The program allows users to:

✅ Create a Person
✅ Create an Employee
✅ Create a Manager
✅ Display stored details
✅ Exit the system

This project is great for beginners learning Python OOP concepts.

🚀 Features
✨ Beginner-friendly Python code
✨ Menu-driven system
✨ Uses OOP concepts
✨ Supports multiple employee types
✨ Displays stored data clearly

🧠 OOP Concepts Used
Concept	Description
Class	Used to define Person, Employee, Manager
Inheritance	Employee inherits from Person
Encapsulation	Private variables like salary
Objects	Objects are created for each role
